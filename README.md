A premium, interactive idle/clicker game built for the web. Players build their network infrastructure, generate currency, and unlock powerful upgrades in a highly polished, Web3-inspired UI with smooth animations.

👉 **[Play Live Demo Here](https://romeototo.github.io/ai-tycoon-rialo-game/)**

## 🚀 Features

- **Core Gameplay Loop:** Tap to generate coins, purchase automated generators, and scale your network.
- **Premium UI/UX:** Dark mode glassmorphism interface with satisfying CSS animations, particle effects, and dynamic node movements.
- **Fever Mode Mechanics:** Build up a combo meter to trigger a high-reward "Fever Mode" with intense visual feedback.
- **Progression System:** Multiple tiers of upgrades with scaling costs to keep long-term retention.
- **Persistent Save System:** Uses `localStorage` to save player progress automatically.
- **Responsive Design:** Fully playable on both Desktop and Mobile devices.

## 🛠️ Tech Stack

- **HTML5 & Vanilla JavaScript (ES6+)** - Zero dependencies, fast execution.
- **Vanilla CSS3** - Custom keyframe animations, flexbox/grid layouts, and glassmorphism UI.
