เกมแนว Idle/Clicker ระดับพรีเมียมบนเว็บ ผู้เล่นจะได้สร้างโครงสร้างพื้นฐานเครือข่าย สร้างรายได้ และปลดล็อกการอัปเกรดที่ทรงพลังผ่าน UI ที่ได้รับการขัดเกลาอย่างดีในสไตล์ Web3 พร้อมแอนิเมชั่นที่ลื่นไหล

👉 **[เล่นเกมที่นี่](https://romeototo.github.io/ai-tycoon-rialo-game/)**

## 🚀 ฟีเจอร์เด่น

- **Core Gameplay Loop:** คลิกเพื่อสร้างเหรียญ, ซื้อเครื่องผลิตอัตโนมัติ และขยายเครือข่ายของคุณ
- **Premium UI/UX:** อินเทอร์เฟซ Dark Mode สไตล์ Glassmorphism พร้อมแอนิเมชั่น CSS ที่น่าดึงดูด, เอฟเฟกต์อนุภาค (Particle) และการเคลื่อนที่ของโหนดแบบไดนามิก
- **Fever Mode Mechanics:** สะสมคอมโบเพื่อเปิดใช้งาน "Fever Mode" ที่ให้รางวัลสูงพร้อมเอฟเฟกต์ภาพสุดตระการตา
- **Progression System:** ระบบอัปเกรดหลายระดับพร้อมต้นทุนที่เพิ่มขึ้นตามความยากเพื่อความสนุกในระยะยาว
- **Persistent Save System:** ใช้ `localStorage` เพื่อบันทึกความคืบหน้าของผู้เล่นโดยอัตโนมัติ
- **Responsive Design:** รองรับการเล่นอย่างสมบูรณ์แบบทั้งบนคอมพิวเตอร์และมือถือ

## 🛠️ เทคโนโลยีที่ใช้

- **HTML5 & Vanilla JavaScript (ES6+)** - ไม่พึ่งพา Library ภายนอก รันได้รวดเร็ว
- **Vanilla CSS3** - แอนิเมชั่น Keyframe แบบกำหนดเอง, เลย์เอาต์ Flexbox/Grid และ UI แบบ Glassmorphism
